1. Difference between the Div and span tag

Div tag

  • Div tag is a Block element
  • It is used to style the section
  • It is mostly used for structuring and styling the web pages
  • It occupies the space of the width of the web pages

Span tag

  • Span tag is a Inline element
  • It is used to style the words which want to highlight the webpages
  • It is used in the div tag
  • It occupies the space based on the content width

4. Difference between the XML and HTML

XML

  • XML means Extensible Markup Language
  • It is aslo a Markup language like HTML
  • it is used to store the data and transport the data with .xml
  • XML tags are not predefined tags like HTML

HTML

  • HTML means Hyper Text Markup Lnaguage
  • It is used to create the web pages
  • It acts like a skeleton to the web pages
  • HTML tags are predefined tags
  • It is also used to store the data with .html extension
